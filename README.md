# youtube-clone

demo: https://youtube-clone-f0gtdxund-jordantseng.vercel.app/

## Techstack

- TypeScript
- Next.js 13.4
- Tailwind CSS

## Features

- Infinite scrolling using intersection observer
- Image optimization using Image component provided by Next.js
- Dark / light mode
- RWD
